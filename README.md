# Возможность расширения
Расширение производится за счёт добавления новых методов составления отчёта.
В коде на примере метода-заглушки показано, по какому принципу можно добавить новые отчёты. Все методы будут принимать словарь с "сырыми" данными и возвращать  список со словарями для вывода в консоль и создания csv-отчёта. 
В зависимости от имени отчёта выбирается метод обработки данных.
